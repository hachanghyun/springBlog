### 개념정리

#### IOC
    객체의 생성과 관리를 개발자가 하는 것이 아니라 프레임워크가 대신하는 것

#### DI
    외부에서 객체를 주입받아 사용하는 것

#### AOP
    프로그래밍을 할 때 핵심 관점과 부가 관점을 나누어서 개발하는 것
